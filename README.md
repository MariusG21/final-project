# E-book App

Aplicație full-stack.

## Cerințe

- Node.js
- Git

# 

## Clonare

```bash
git clone https://github.com/MariusG21/final-project.git
```

Structura proiectului:

```txt
final-project/
├── bibliotek
└── bibliotek-backend
```

## Deschiderea proiectului

Deschide proiectul în două terminale separate:

### Terminal 1 — Frontend

```bash
cd /final-project/bibliotek
npm install
npm run dev
```

### Terminal 2 — Backend

```bash
cd /final-project/bibliotek-backend
npm install
npm start
```

Aplicația rulează la:

-backend: http://localhost:3000
-frontend: http://localhost:5173
